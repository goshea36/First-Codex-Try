# Exam FM Study Planner

Simple single-page web app to create an FM study plan.

## What it includes
- Inputs for exam date, number of topics, hours available per week, and study days per week.
- A built-in checklist of Exam FM subjects/topics with completion checkboxes and a Finished Topics list.
- Calculations for:
  - Days until exam
  - Total study hours available
  - Suggested daily study time (weekly hours รท study days per week)
  - Topics left to study
- Weighted topic allocations based on relative topic importance, automatically redistributed across remaining unfinished topics.

## Run locally
Open `index.html` in your browser.
